## Description

### openCV

* expansion: 画像を反復しながら拡張する

* flip: 画像を左右に反転させる

* face: 顔を検知する

* grayscale: 画像をグレースケール化する

* resize: 画像のサイズを変更する

* webcam: カメラの映像を加工する
  * webcam2manga.py: カメラの映像を漫画風に変更する
